# Rectangle

Rectangle shape.


## Installation

    npm install @pencil.js/rectangle


## Examples

```js
    import Rectangle from "@pencil.js/rectangle";
    
    const width = 200;
    const height = 100;
    const options = {
        fill: "red",
        stroke: "#000"
    };
    let rectangle = new Rectangle(aPosition, width, height, options);
```
